# -SuperStore-Sales-Analysis-Dashboard-Power-BI
End-to-end retail sales analysis on 5,900+ transactions (2019–2020) built entirely in Power BI. Covers data transformation via Power Query, DAX-powered KPIs, and 4 interactive dashboards across sales performance, regional trends, product profitability, and customer segmentation, plus a 15-day sales forecast using Power BI's built-in ETS model.
